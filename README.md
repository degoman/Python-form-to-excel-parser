# form to excel parser

This is a simple python script for parsing specific form into an exisiting excel form.
Script comes with a simple UI. 

Current form format:
```
Z
alice@gmail.com
Názov formulára
Prihlasovací formulár
Meno, priezvisko a dátum narodenia (dd mm rrrr) dieťaťa
Alice Bob 1.1.2010
Meno a priezvisko zákonného zástupcu dieťaťa
Catharine Bob 
Telefónne číslo na zákonného zástupcu dieťaťa - ktoré používate pre whats app aplikáciu
0903000101
Adresa zákonného zástupcu dieťaťa
Main 42, 00055 CityName 
Vyberte si deň kurzu
Tuesday 14:00 - 15:30
Názov školy
St.Learner school 
Trieda (príklad: 3 c)
1.A.
```
